# Clockcoin

## Required Tools

- MetaMask (Chrome Plugin)
- Ethereum Remix ()

## Setup

```bash
# Install required packages
yarn

# Start development blockchain
yarn rpc

# You can use the private keys you see in the console to add accounts on MetaMask
```
